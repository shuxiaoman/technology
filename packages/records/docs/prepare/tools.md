# 开发工具

## chrome

> 开发必备

### 插件推荐

> 最好和 google 账户绑定，翻墙后自动同步

- FeHelper（可快速查看 json，不用进入 ide 或在线 json 网站）
- Proxy SwitchyOmega（根据域名，自动判断是否用梯子访问）
- Xswitch、react-tools、vue-tools 根据需求自行安装
